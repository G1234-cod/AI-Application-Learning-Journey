## 实践规划与阶段产出

本项目分为四个核心工程阶段，逐步完成从 API 基础调用到复杂智能体的开发。

- [ ] **Phase 1: API 调用与环境集成**
  - 熟悉 HTTP 接口交互、JSON 解析及大模型参数控制。
  - 产出：基于命令行的文本处理工具。
- [ ] **Phase 2: RAG 知识库构建**
  - 掌握文本切分、向量化 (Embedding) 及向量数据库操作。
  - 产出：基于本地文档的问答助手。
- [ ] **Phase 3: 全栈 AI 应用**
  - 结合 FastAPI/Spring Boot 封装后端，实现流式输出 (SSE)。
  - 产出：部署于云服务器的 Web 智能对话交互端。
- [ ] **Phase 4: Agent 智能体开发**
  - 探索 Function Calling (工具调用) 与自动化任务流。
  - 产出：具备外部搜索能力的分析辅助程序。
